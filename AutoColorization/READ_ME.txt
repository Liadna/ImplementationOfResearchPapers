
This program auto colorize images using a trained CNN based on VGG-16 net.

----------------------------------------

1. Save the project on Desktop

2. Put images in image directory 

3. run the project

 	-> The gray pictures will be save in Gray directory

	-> The results  will be save in resutls diractory

4. Open results directory to see the results
